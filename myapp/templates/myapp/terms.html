1. Introduction

This Penetration Testing Simulation Platform is an educational tool designed to allow students to practice ethical hacking techniques in a controlled and supervised environment. By using this Platform, students contribute to real-world causes, such as identifying and enhancing security vulnerabilities in software systems for nonprofits or public organizations.

2. Purpose and Scope

The Platform is strictly for educational purposes and ethical hacking simulations. It is intended to:

  • Provide hands-on experience in identifying software vulnerabilities.
  • Raise awareness of cybersecurity risks.
  • Contribute to enhancing the security of nonprofits or public organizations through responsible vulnerability reporting.

3. User Responsibilities

Users of the Platform must:

  • Only conduct activities within the controlled environment of the Platform.
  • Refrain from using any techniques or methods learned on the Platform for illegal or unethical purposes outside the Platform.
  • Maintain the confidentiality of any security vulnerabilities discovered during testing.

Users must adhere to strict ethical standards while using the Platform. This includes:
